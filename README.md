# easy_meyqli

## 1. 简介

**easy_mysqli** 是对 **mysqli** 的封装，提供了常用的增删改查功能，主要是面向 PHP 入门的初学者使用的，在不使用框架的情况下，可以快速操作数据库。

**easy_mysqli** 的特点有：

* 强数据类型
* 使用语句预处理
* 事务提交和回滚
* 直接引入即可使用

## 2. 基本功能

**1. 查询**

* 单键值查询
* 多行键值查询
* 使用 *WHERE* 的多行键值查询

**2. 增加**

* 新增单行数据

**3. 修改**

* 通过主键修改其他字段

**4. 删除**

* 通过主键删除某行数据

## 3. 计划实现的功能

1. 使用排序的查询
2. 针对 date、time 类型的条件查询
